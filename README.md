# Windows Security Checker & Toolkit

Набор интерактивных инструментов (TUI) на базе PowerShell для автоматизации аудита безопасности, контроля соответствия требованиям ФСТЭК России (РД АС / РД СВТ), сбора состава программного обеспечения, аудита подключений накопителей из реестра и запуска сторонних диагностических утилит (ScanOval, USBDeview, HWInfo, WinAudit).

Комплекс разработан для упрощения проведения аттестационных проверок и аудита информационной безопасности на рабочих станциях и серверах под управлением ОС Windows в закрытых и изолированных контурах.

---

## 🚀 Основные возможности

### 1. 📊 Комплексный аудит безопасности и комплаенс-контроль
- **Сбор политик учетных записей**: опрашивает параметры локальных политик паролей и блокировок (длина, срок действия, порог попыток входа, авторазблокировка) с использованием `secedit.exe` и SAM через ADSI (для языковой независимости).
- **Анализ параметров ядра и сессий**: проверяет состояние защиты LSA (`RunAsPPL`), очистку файла подкачки при выключении (`ClearPageFileAtShutdown`), таймауты блокировок экрана при простое и параметры выполнения скриптов PowerShell.
- **Контроль соответствия классам защищенности**: оценка настроек на соответствие требованиям РД АС и РД СВТ для классов (`3Б, 2Б, 1Д, 1Г, 3А, 2А, 1В, 1Б, 1А`). Параметры считываются из настраиваемого конфигурационного файла `service/compliance_standards.conf`.
- **Генерация отчетов**:
  * Подробный текстовый лог соответствия с рекомендациями по устранению несоответствий (`report/compliance_report_[КЛАСС]_[ДАТА].txt`).
  * Единый автономный интерактивный HTML-отчет (`report/security_report.html`) с темной/светлой темой, графическим индикатором уровня защищенности и сводными таблицами.

### 2. 🔌 Сбор истории подключенных USB-устройств
- Производит опрос защищенных веток системного реестра `HKLM\SYSTEM\CurrentControlSet\Enum\USBSTOR` и `Enum\USB`.
- Формирует полную структурированную таблицу всех съемных накопителей, когда-либо подключавшихся к АРМ, включая: дружественное имя устройства, серийный номер, наименование драйвера и описание.

### 3. 📦 Состав программного обеспечения (ПО)
- Сканирует ветки реестра `Uninstall` в разделах `HKEY_LOCAL_MACHINE` (как 64-битные, так и 32-битные подразделы `Wow6432Node`) и `HKEY_CURRENT_USER` (установки в профиле пользователя).
- Формирует список установленного ПО (имя, версия, издатель, дата установки) и сохраняет его в файл `report/installed_software.txt` с последующей интеграцией в HTML-панель (поддерживающую живой поиск).

### 4. 🛡️ Аудит СЗИ (Dallas Lock / Secret Net Studio) и антивирусов
- **Выбор целевого СЗИ**: интерактивный выбор ожидаемого к установке СЗИ от НСД (**Dallas Lock** или **Secret Net Studio**) для последующей проверки.
- **Интеграция СЗИ**: проверка присутствия файлов, путей установки, веток реестра и активности всех служб защиты (например, `DlSrv` для Dallas Lock или служб с префиксом `sns` для Secret Net Studio).
- **Мониторинг антивирусного ПО**: проверка активности Windows Defender, Kaspersky Endpoint Security и Dr.Web.
- **Поиск дистрибутивов**: сканирование подключенных съемных накопителей на наличие установочных пакетов СЗИ и антивирусов.

### 5. 🔍 Интеграция диагностических утилит
- Автоматически находит и запускает встроенные утилиты из папки `tools/`:
  - **WinAudit** — детальный отчет по программно-аппаратной части (`report/winaudit_report.html`).
  - **USBDeview** — отчет по состоянию портов и USB-устройств (`report/usbdeview_report.html`).
  - **HWInfo** — характеристики оборудования АРМ (`report/hwinfo_report.txt`).
  - **ScanOval** — проведение сканирования уязвимостей ФСТЭК России по базам OVAL.
- Интегрирует ссылки на сформированные сторонние отчеты в главный HTML-отчет.

### 6. 🛠️ Менеджер зависимостей (Repo Manager)
- Проверяет подключение к интернету и БДУ ФСТЭК.
- Автоматически скачивает портативные версии утилит (WinAudit, USBDeview, HWInfo) и базы данных уязвимостей ФСТЭК напрямую в нужные папки проекта.
- В изолированных сетях позволяет выполнить поиск и автоматическое копирование бинарных файлов со съемных накопителей (CD-ROM, USB-диски) в рабочую папку `tools/`.

## 🛠️ Системные требования

- **Операционная система**: Windows 7 / 8 / 10 / 11, Windows Server 2012R2 и выше.
- **Среда выполнения**: PowerShell 5.1 и выше.
- **Права доступа**: Запуск строго от имени Администратора (требуется UAC-авторизация).

---

## 🚀 Быстрый старт

1. Перенесите каталог проекта на целевую машину (например, с помощью USB-накопителя).
2. Щелкните правой кнопкой мыши по файлу **`hub.bat`** в корне проекта и выберите **«Запуск от имени администратора»**.
3. Подтвердите запрос UAC. Откроется интерактивное консольное меню управления.

## 📂 Структура проекта

```
.
├── hub.bat                      # Файл автозапуска хаба с правами администратора
├── hub.ps1                      # Главный централизованный хаб управления (TUI)
├── readme.txt                   # Краткая инструкция по запуску
├── about.txt                    # Техническая справка по работе модулей
├── README.md                    # Описание проекта для репозитория
├── scripts/
│   ├── security_checker.ps1     # Скрипт аудита безопасности, сбора ПО и USB
│   ├── scanoval_checker.ps1     # Скрипт запуска сторонних утилит (WinAudit, ScanOval...)
│   ├── antivirus_installer.ps1  # Проверка антивирусной защиты, выбор и аудит СЗИ
│   └── repo_manager.ps1         # Управление зависимостями (скачивание, копирование)
├── service/
│   ├── report_generator.ps1     # Сборщик интерактивного HTML-отчета аудита
│   └── compliance_standards.conf # Настройки эталонов соответствия классам защищенности
├── report/                      # Каталог сохранения всех отчетов
└── tools/                       # Папка размещения сторонних исполняемых файлов утилит
```

## 📝 Лицензия

Проект предназначен для внутреннего использования и автоматизации процессов администрирования. Права на сторонние утилиты (WinAudit, USBDeview, HWInfo, ScanOVAL) принадлежат их соответствующим авторам и правообладателям.
