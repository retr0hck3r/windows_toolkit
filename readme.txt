========================================================================
             ИНСТРУКЦИЯ ПО ЗАПУСКУ И ИСПОЛЬЗОВАНИЮ (WINDOWS)
========================================================================

Данный программный комплекс предназначен для аудита безопасности систем Windows.
Все операции должны производиться от имени Администратора.

------------------------------------------------------------------------
ЗАПУСК ИНТЕРАКТИВНОГО МЕНЮ (ХАБА)
------------------------------------------------------------------------
Запустите центральную панель управления:
  - Дважды щелкните по файлу `hub.bat` в корне проекта.
  - Согласитесь на запрос контроля учетных записей (UAC) для предоставления прав Администратора.
  
(Файл `hub.bat` автоматически настроит обход политик выполнения скриптов PowerShell (`-ExecutionPolicy Bypass`) для текущей сессии).

После запуска на экране появится интерактивное меню с выбором действий:

1) Аудит и СЗИ
   - Выполняет комплексную проверку локальной системы, включая контроль соответствия требованиям РД АС и РД СВТ для выбранного класса защищенности.
   - Считывает эталонные параметры (длина паролей, блокировки, LSA, Pagefile и т.д.) из настраиваемого файла `service/compliance_standards.conf`.
   - Проверяет статус и работоспособность выбранного СЗИ (Dallas Lock или Secret Net Studio).
   - Опрашивает реестр для составления полной истории подключенных USB-накопителей (USBSTOR).
   - Результаты сохраняются в:
     * report/security_report.html (интерактивный графический HTML-отчет)
     * report/installed_software.txt (список установленного ПО)
     * report/compliance_report_[КЛАСС]_[ДАТА_ВРЕМЯ].txt (текстовый лог аудита соответствия)

2) Внешние утилиты (ScanOval, USBDeview, HWInfo, WinAudit)
   - Проверяет наличие сторонних диагностических утилит в папке `tools/`.
   - Запускает автоматизированный сбор данных через:
     * WinAudit — генерирует подробный снимок системы в `report/winaudit_report.html`.
     * USBDeview — экспортирует детализированный список USB в `report/usbdeview_report.html`.
     * HWInfo — выгружает спецификации оборудования в `report/hwinfo_report.txt`.
     * ScanOval — проводит сканирование уязвимостей по базам OVAL ФСТЭК России.
   - Интегрирует ссылки на сформированные отчеты в главный HTML-отчет.

3) Установка антивирусного ПО и выбор СЗИ
   - Позволяет выбрать целевое СЗИ (Dallas Lock, Secret Net Studio или Без СЗИ) для последующих комплаенс-проверок.
   - Анализирует состояние текущей защиты (Windows Defender, Kaspersky Endpoint Security, Dr.Web).
   - Осуществляет поиск инсталляторов защитного ПО (kes*.exe, kes*.msi, drweb*.exe, dl*.exe, sns*.exe) на подключенных съемных накопителях.

4) Инструменты (Менеджер зависимостей)
   - Проверяет наличие всех необходимых утилит диагностики в папке `tools/`.
   - При подключении к сети Интернет позволяет автоматически скачать портативные версии USBDeview, WinAudit, HWInfo и XML-базу определений уязвимостей ФСТЭК.
   - При автономной работе выполняет поиск и копирование недостающих бинарных файлов со съемных носителей (флешек, CD/DVD дисков).

0) Выход
   - Завершает работу хаба управления.
